┌─────────────────────────────────────────────────────────────┐
│ OPCION 1: Usar VideoProcessor con Python (RECOMENDADO)     │
└─────────────────────────────────────────────────────────────┘

  Esta opcion instala automaticamente ultralytics.

  PASOS:

  1. Doble click en: TrafficAnalysisSystem.bat

  2. Seleccionar opcion: [2] VideoProcessor Python

  3. Esperar a que instale ultralytics (solo primera vez, ~2 min)

  4. Usar normalmente

  ✓ Instalacion automatica
  ✓ Siempre funciona
  ✓ Usa CPU (sin CUDA)


┌─────────────────────────────────────────────────────────────┐
│ OPCION 2: Instalar Ultralytics Manualmente                  │
└─────────────────────────────────────────────────────────────┘

  Si prefieres usar el ejecutable VideoProcessor.exe:

  1. Abrir Command Prompt (CMD)

  2. Ejecutar:
     pip install ultralytics

  3. Esperar (~2-3 minutos)

  4. Usar VideoProcessor.exe normalmente

█ PREGUNTA: ¿Por que pasa esto?
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  PyInstaller debe empaquetar todas las dependencias en el .exe,
  pero ultralytics + torch son muy pesados (~500 MB) y a veces
  la compilacion no se completa correctamente.

  La opcion de Python instala las dependencias directamente y
  evita este problema.
